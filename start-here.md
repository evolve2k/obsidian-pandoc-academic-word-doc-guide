# Start Here

## Useful Shortcuts
`A guide to the shortcuts in use in Obsidian.`

Ctrl + Shift + m - Add inline markdown citation
Ctrl + Shift + o - Add Literature Note
Ctrl + Shift + d - Open Daily Note for today

Make an inline link:   `[[` type two opening square brackets to bring up the Obsidian pages lookup.
